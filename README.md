# Bitcoin Era - Cryptocurrency Trading Platform

A professional, responsive cryptocurrency trading platform website built with pure HTML, CSS, Bootstrap, and JavaScript.

## Features

- **Responsive Design**: Mobile-first approach, works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Real-time Forms**: Contact form and signup modal with validation
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Bootstrap Framework**: Built with Bootstrap 5 for robust components
- **Font Awesome Icons**: Professional icons throughout the site
- **JavaScript Interactivity**: Form handling and animations
- **GitHub Pages Ready**: Deploy directly to GitHub Pages

## Getting Started

### Option 1: Clone and Deploy to GitHub Pages

1. Create a new repository on GitHub named `bitcoin-era-website`
2. Clone the repository:
   \`\`\`bash
   git clone https://github.com/Dremorsamuel3/bitcoin-era-website.git
   cd bitcoin-era-website
   \`\`\`

3. Add the files to your repository

4. Push to GitHub:
   \`\`\`bash
   git add .
   git commit -m "Initial commit: Bitcoin Era website"
   git push origin main
   \`\`\`

5. Enable GitHub Pages in repository settings:
   - Go to Settings > Pages
   - Select `main` branch as source
   - Your site will be live at: `https://dremorsamuel3.github.io/bitcoin-era-website`

### Option 2: Local Development

1. Open `index.html` in your browser
2. Make changes to `styles.css` or `script.js`
3. Refresh your browser to see updates

### Navigation
- Sticky navigation bar

- Active link highlighting
